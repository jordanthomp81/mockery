Clean User Experience

I wanted to create an interface that was clean, modern, and intuitive

to use. This simple application will allow you to painlessly create

new mockups, and share them with the world.

Features

The majority of wireframe tools are bloated and complicated to use. My goal

was the boil down the features into an easy to use interface.

Easily Create Wireframes

With mockery you can quickly and easily get started with wireframing. Our tools let you create, resize, and delete an array of shapes and elements to create your perfect layout. To get started simply use the toolbar on the left to create the elements that you need.

Try Mockery

Share Your Mockery

The design proccess always relies on the collaboration of users and designers. Mockery makes sending your mockup incredibly easy. For every wireframe that you create, you will recieve a public project url that you can send to anyone.

Try Mockery

Snap Grid

It was vital for this application to be incredibly easy to get up and running with. Mockey uses a 20x20 grid that allows you to drag, snap , reorder, and resize with complete precision. This allows you to focus on designing your product, rather than worrying about spacing or margins.
